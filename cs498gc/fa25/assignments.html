Assignments

Important: All assignments must be submitted via Gradescope. Late submissions will be penalized 10% per day. Each student has 3 late days to use throughout the semester.

Coding Exercises (45% of grade)

Coding exercises are hands-on programming assignments where you will implement key algorithms for mobile robotics using ROS2. These assignments involve processing real sensor data and implementing state estimation algorithms.

Coding Exercise 1: Wheel Odometry and ROS2 Basics

Release Date: September 16, 2025

Due Date: September 30, 2025 (11:59 PM)

Weight: 15% of total grade

Objectives:
  • Implement wheel odometry computation from encoder data
  • Create ROS2 nodes for publishing odometry messages
  • Handle coordinate transformations using tf2
  • Visualize robot trajectory in RViz2
Deliverables:
  • Python implementation of odometry node
  • Output trajectory files for autograding
  • Brief report explaining your approach

Final Project (20% of grade)

Final Project: Advanced Mobile Robot System

Release Date: November 11, 2025

Checkpoint Due: December 2, 2025

Final Submission: December 11, 2025

Presentations: December 9-11, 2025

Weight: 20% of total grade (25% for 4-credit students)

Project Options:
  1. Visual-Inertial Odometry: Implement VIO using camera and IMU data
  2. Multi-Robot Localization: Coordinate multiple robots for collaborative mapping
  3. Learning-Based Navigation: Apply deep learning to navigation problems
  4. SLAM Implementation: Implement a complete SLAM system from scratch
  5. Custom Project: Propose your own project (requires approval)
Requirements:
  • Can be done individually or in groups of 2-3
  • Must use ROS2 and real/simulated sensor data
  • Include comprehensive evaluation and comparison
  • Final presentation and demonstration required
  • 4-credit students must include additional research component
Deliverables:
  • Project proposal (1 page)
  • Midpoint checkpoint with preliminary results
  • Final code repository with documentation
  • Technical report (6-8 pages)
  • 15-minute presentation
