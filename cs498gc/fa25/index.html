CS498GC: Mobile Robotics

Fall 2025

Course Description

This course will introduce students to foundational principles of mobile robotics with a view towards developing software for robots. Topics covered will be dynamic modeling, coordinate transformations, principles of operations of different sensors, sensor fusion algorithms including Kalman filters, introduction to Simultaneous Localization and Mapping, and introduction to feedback control for robotics.

Topics covered include:

  • Introduction to Mobile Robotics and ROS2
  • Coordinate transformations and quaternions
  • Dynamics of mobile robots (holonomic and nonholonomic)
  • Principles of robot motion control
  • Multi-sensor proprioceptive sensor fusion
  • GPS-INS sensor fusion with Extended Kalman Filters
  • Symbolic exteroceptive perception (LiDAR, cameras)
  • Localization, mapping, and SLAM
  • Frontiers in mobile robotics

Key Learning Outcomes

By the end of this course, students will be able to:

  • Implement and debug ROS2 nodes for robot control and perception
  • Design and implement sensor fusion algorithms using Extended Kalman Filters
  • Process and integrate data from multiple sensors (IMU, GPS, encoders)
  • Understand and apply coordinate transformations in robotics
  • Develop complete mobile robot navigation systems
  • Evaluate and optimize robot localization accuracy

Prerequisites

  • MATH 221, MATH 225, and MATH 285
  • STAT 400 or equivalent
  • CS 125, CS 225 or equivalent
  • Programming experience in Python

Course Format

The course consists of lectures, coding exercises, problem sets, and a final project. Students will work with real sensor data from mobile robots and implement algorithms that process this data to achieve accurate localization and navigation.

Important: This course requires significant programming work. Students should be comfortable with Python and willing to learn ROS2.
